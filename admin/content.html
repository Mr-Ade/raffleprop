PDF Accessibility Requirements (WCAG 2.1 AA)
  • PDFs must be tagged — do not upload scanned image-only PDFs (invisible to screen readers)
  • Use Word / Google Docs → Export as PDF, or Adobe Acrobat with tagging enabled
  • Set a document title and language in PDF properties before uploading
  • Run Adobe Acrobat Accessibility Check (Tools → Accessibility → Full Check) before uploading
  • External URL option: host PDFs on Google Drive or Dropbox — ensure sharing is set to "Anyone with the link"
